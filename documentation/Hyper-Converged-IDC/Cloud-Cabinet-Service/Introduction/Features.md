# 产品功能

**资源展示**

   控制台展示客户已正式分配的机柜资源，显示机柜基本信息，开通状态等。

   控制台展示客户托管设备资源，显示设备基本信息，上架机柜等。
  
   控制台展示客户购买的公网IP，带宽资源，显示IP地址信息，共享带宽信息。
   
 **资源监控**

   客户可查看已购买的带宽的实时流量。

   系统提供1小时，6小时，12小时，24小时以及90天内的流量数据查询。

   客户可为流量资源进行报警设置。

**线上申请**
    
   支持客户在线提交运维服务申请。

   工单申请进程实时跟踪，了解处理进度

**运维服务**
    
   提供基本运维服务，帮助客户进行设备异动，设备发送接收等服务。

   按需提供增值服务，帮助客户维护网络，设备，系统支持。
