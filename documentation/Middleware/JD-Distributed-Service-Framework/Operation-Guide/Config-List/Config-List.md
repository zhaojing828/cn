#  配置管理列表
配置管理，提供给用户对应用进行配置的服务。每个配置中，可以存在多个版本，并且每个版本可发布在同一应用所在的不同部署组中。

### 入口：
-  互联网中间件>微服务平台>配置管理 
-  互联网中间件>微服务平台>服务注册>注册中心>配置管理
