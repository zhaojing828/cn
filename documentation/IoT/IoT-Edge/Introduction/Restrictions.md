# 限制说明

您可以快速创建并部署使用物联网边缘计算产品 IoT Edge，但有如下限制请在使用中注意。

|        项目        |    数量 |                说明                 |
| :----------------: | ------ | :---------------------------------: |
|   创建Edge节点数   | ≤10000 | 您单个账户下最多可以创建10000个Edge节点 |
| 单Edge节点子设备数 | ≤100   |  单个Edge节点下最多连接管理100个子设备  |
|    边缘计算模块    | ≤5     | 单个Edge节点下最多可部署5个边缘计算模块 |
