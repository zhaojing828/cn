# 产品性能

云文件服务性能参数如下：


| **指标**    | **类型**    | **min** |**备注**                        |
| ----------- | ----------- | ------- |------------------------------- |
| 带宽吞吐    | write       | 50MB/s  |根据文件系统容量，增幅0.1MB/s * V(GB)             |
|             |  read       |  50MB/s   |根据文件系统容量，增幅0.1MB/s * V(GB)          |
| 时延        | write       | 10ms    |IO size为4K，在有压力下的时延 |
|             |  read      | 10ms     |IO size为4K，在有压力下的时延 |
