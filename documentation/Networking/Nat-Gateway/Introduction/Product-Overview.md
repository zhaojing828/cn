# 产品概述

NAT网关(NAT Gateway)是一款托管的VPC公网网关，作为VPC环境下公网流量出口，为用户提供更便捷，可靠的Internet访问。

作为京东云全自研的产品，具备以下特点：
* 完全托管 京东云完全托管，用户无需维护
* 高性能 京东云自研的分布式NAT网关，提供最大4Gbps的吞吐量和最大1000万并发连接数
* 高可用 提供AZ级别高可用
* 按需购买 NAT网关规格和公网IP带宽均可以随时调整，轻松应对业务变化
